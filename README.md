# Financial_Loan_Analysis
# Project Overview
This project provides a comprehensive analysis of loan data using Excel Dashboards. The aim is to monitor loan performance, identify trends, and track key financial metrics to support data-driven decision-making.

The project includes three interactive dashboards:

# Dashboard 1: Summary
# Dashboard 2: Overview
# Dashboard 3: Details

# üìä Dashboard 1: Summary
Key Performance Indicators (KPIs)

# Total Loan Applications:
Calculates the total number of applications received and monitors Month-to-Date (MTD) and Month-over-Month (MoM) performance.
# Total Funded Amount:
Measures the total amount of loans disbursed and tracks MTD and MoM changes.
# Total Amount Received:
Tracks repayments received from borrowers to assess cash flow performance.
# Average Interest Rate:
Computes the average lending rate across all loans and observes MoM fluctuations.
# Average Debt-to-Income Ratio (DTI):
Evaluates borrower financial health by measuring average DTI ratios and monitoring trends.
